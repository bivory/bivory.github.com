Runner†

This is day one of my attempt to complete the #onegameamonth challenge. The game is implemented in C#, which I'm learning as I go, using Unity3d, which I'm also learning as I go.

State†

At this point Runner is a more or less a fork of the awesome Runner tutorial written by Catlike Coding. My version isn't quite as awesome though. In fact it's barely playable. The controls are sluggish, the game doesn't detect when the player dies, there's no way to restart the game other than reloading the page and good luck surviving more that four or five platforms.
